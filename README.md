## 🔹 Structure Explanation

In both folders (**Oracle** and **PostgreSQL**):

- Each problem is stored as a separate `.sql` file  
- File names correspond directly to the problem titles  

### ✅ Benefits of this approach:
- 📌 Improves navigation  
- 📌 Makes comparison between Oracle and PostgreSQL easier  
- 📌 Provides a clean and modular project structure
